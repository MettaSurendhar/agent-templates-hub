# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) in spirit:

- Be respectful and constructive in issues, PRs, and discussions.
- No harassment, personal attacks, or spam.
- Disagreements about template design/structure are fine and expected — keep them focused on the work.

Violations can be reported by opening an issue tagged `conduct` or contacting the maintainer directly via GitHub.
